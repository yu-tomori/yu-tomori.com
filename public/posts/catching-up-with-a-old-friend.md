# 何年もあっていない旧友に会う

何回か失敗して学んだことを書き留める。

何年もあっていない旧友と久しぶりに会うとき、以下のことを認識するのが大事。

昔は心を通じ合った仲でも、会わない間にお互いが異なる時間と空間を過ごしていること。ゆえに、これから会うあの人は確実に変わっていること。

彼（彼女）が高校時代に好きだった音楽を今も好んで聴いているとは限らない。
彼（彼女）はもうバンドやアイドルに興味がないかもしれない。私が彼（彼女）の好きなものを決め込んで、そのつもりで話すと、彼（彼女）はうんざりするかもしれない。

目の前の彼（彼女）は、細胞も記憶も入れ替わった別人であるという前提を置くこと。新しい他人に出会って、新しい関係性を築く姿勢でさえいれば、旧友に久しぶりに会うことほど面白いことはない。

# 過去をみること

とはいえ一度心を通じ合ったという相互認識は再び新しい関係性を築くにあたって強力な触媒となる。

共に味わった体験を一緒に思い出して「そういえばそうだったよね」とか、今だから言えることを「おれはあのとき実はこうおもってた」とか話す。このとき、当時の相手について、当時の場について、より多面的に、より鮮明に、知ることになる。それは、あの時教えてくれなかったことを今知ったから、だけではなくて、私の感性もその時より豊かになっているからである。

&nbsp;

あなたは、当時のあなたについてもより客観的な姿を知ることとなる。相手は当時の自分の客観的な様子を記憶にとどめている。相手の声とか表情によって封が切られ、記憶が呼び覚まされ、芋づる式にあんなことこんなことが頭に浮かんでくる。

相手が持っている鏡に映っている自分の姿は、大抵自分の記憶とは違っている。なぜなら、自分の記憶は自己解釈により捏造（良く言えばストーリー化）されている。当時の自分は自己像を捏造しないとやってけない心持ちだったかもしれない。又は、自己のストーリーを捏造することで未来に向かう自分を鼓舞したのかもしれない。そんな自分はもはや対象化され、鑑賞の対象となっている。今だからありのままの自分を客観的に捉えられるし、受け入れられる、かもしれない。もしくは、もっと時間を要するかもしれない。

いずれにしても、相手が話す客観的な自分の姿を何年越しかに鑑賞するのはとても面白いことだ。

# これまでをキャッチアップすること

離れてから何があったのかを聞くこと。「あのあとどうした？」、「大学入ってからどうした？」、「どこに住んでたんだっけ？」、「あいつどうしてるかな」とか。熱が入ってきたら「これまでの人生で最も影響を受けた出来事は何？」とかストレートに聞いてもいいかもしれない。相手は、自身のことを再認識しながら、そして新たなストーリーをその場で捏造しながら、話し出すだろう。

自分が知っている最後のスナップショットと、今目の前にいるスナップショットには明確な差分がある。その差分を形成したのは、どんな体験だったのかキャッチアップしよう。

相手の変わったところ、変わらなかったところを認識する。

相手が選ぶ言葉や相手から仕方なく滲み出す調子は、その人にとっての生きた世界の様相であり、感性の具現化された姿である。

その人が信じていること（又は信じていないこと）、希望をいだいていること（又は絶望を感じていること）をありありと表出させる。

言葉には暴露性がある。変化を語る言葉にこそ、その人の変わらない感性が染み付いている。

そこで初めて、その人の変わらない姿を知ることになる。一定期間一緒にいただけでは分からない。一度離れて時間を開けたあとに点をつながないと本質は分からないものである。三角測量するように相手の本質を捉えよう。

自分が相手の持っている鏡を覗き込んでいる時、相手も自分の持っている鏡を覗き込んでいることに気づくだろう。お互いに発見がある良い時間だ。

# これからのこと

相手はどこに向かっているのか、数年も経てばライフステージが変わったり、関わる人が変わったり、住む場所が変わったり。生きていれば大変なことやとんでもなく幸せなことがある。それは覚悟を決めたり踏ん張ったりリスクを取ったりした対価である。

共に未来に向かう同士として覚悟やリスクを共有するほど打ち解けられれば、また数年後に会ったときとんでもなく楽しい場になるだろう。

どんな人でも不確実な未来に自分の身を投げ出している。その人が抱えている不安と期待が入り交じった感情に寄り添い、無条件に応援してあげよう。何年も会っていなかった他人の人生だから、究極自分に関係ないのだけど、他人だからこそ無条件に応援できる。何年も会わない関係性も、特別な存在になりうる。

そうやって相手に寄り添うことは不思議と自分の活力にもなるものである。あいつも何とかやっているからおれももう少し頑張ってみようとか、あいつが新しい環境に行ったから自分も外に出てみようとか、また会うときにこの話しようとか。

&nbsp;

また会おうと言って改札で道端で家で分かれたあと、自分の中に確かな活力がじりじりと音を立てているのを聞こう。

私はどこからきたのか、私は何者なのか、私はどこに行くのか、自分が捏造した新しいストーリーができあがっているかもしれない。リラックスして旧友に語った言葉は、自分に正直なありのままのストーリーであることが多い。それが好きなら受け入れて繰り返し語ろう。でも嫌なら忘れて、数年後にまた鏡をみればいい。

新しい他人に出会って、新しい関係性を築く姿勢でさえいれば、旧友に久しぶりに会うことほど面白いことはないと思う。
